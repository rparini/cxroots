ContoursÔÉÅ

The cxroots module allows the user to specify four different types of contours which are all subclasses of Contour:

  • Circle

  • Rectangle

  • Annulus

  • Annulus Sector

Integrate the function f along the path.

\[\oint_C f(z) dz\]

The value of the integral is cached and will be reused if the method is called with same arguments.

Parameters:

  • f (function) ‚Äì A function of a single complex variable.

  • abs_tol (float, optional) ‚Äì The absolute tolerance for the integration.

  • rel_tol (float, optional) ‚Äì The realative tolerance for the integration.

  • div_max (int, optional) ‚Äì If the Romberg integration method is used then div_max is the maximum number of divisions before the Romberg integration routine of a path exits.

  • int_method ({'quad', 'romb'}, optional) ‚Äì If ‚Äòquad‚Äô then scipy.integrate.quad() is used to compute the integral. If ‚Äòromb‚Äô then Romberg integraion, using scipy.integrate.romberg(), is used instead.

Returns:

The integral of the function f along the path.

complex

class cxroots.AnnulusSector(center: complex, radii: Tuple[float, float], phi_range: Tuple[float, float])[source]ÔÉÅ

A sector of an annulus in the complex plane.

Parameters:

  • center (complex) ‚Äì The center of the annulus sector.

  • radii (tuple) ‚Äì Tuple of length two of the form (inner_radius, outer_radius)

  • phi_range (tuple) ‚Äì Tuple of length two of the form (phi0, phi1). The segment of the contour containing inner and outer circular arcs will be joined, counter clockwise from phi0 to phi1.

Examples

from numpy import pi
from cxroots import AnnulusSector
annulusSector = AnnulusSector(
    center=0.2, radii=(0.5, 1.25), phi_range=(-pi/4, pi/4)
)
annulusSector.show()
