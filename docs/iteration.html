cxroots.iterative_methods.iterate_to_root(x0: complex, f: Callable[[complex | float], complex], df: Callable[[complex | float], complex] | None = None, step_tol: float = 1e-12, root_tol: float = 1e-12, max_iter: int = 20, refine_roots_beyond_tol: bool = False, callback: Callable[[complex, complex, complex, int], bool] | None = None) → complex | None[source]ÔÉÅ

Starting with initial point x0 iterate to a root of f. This function is called during the rootfinding process to refine any roots found. If df is given then the Newton-Raphson method, newton(), will be used, otherwise Muller‚Äôs method, muller(), will be used instead.

Parameters:

  • x0 (complex) ‚Äì An initial point for the iteration.

  • f (function) ‚Äì Function of a single variable which we seek to find a root of.

  • df (function, optional) ‚Äì The derivative of f.

  • step_tol (float, optional) ‚Äì The routine ends if the step size, dx, between sucessive iterations satisfies abs(dx) < step_tol and refine_roots_beyond_tol is False.

  • root_tol (float, optional) ‚Äì A root, x, is only returned if abs(f(x)) < root_tol, otherwise None is returned

  • max_iter (int, optional) ‚Äì The routine ends after max_iter iterations.

  • refine_roots_beyond_tol (bool, optional) ‚Äì If True then the routine ends only once the error of the previous iteration, x0, was at least as good as the current iteration, x, in the sense that abs(f(x)) >= abs(f(x0)), and the previous iteration satisfied abs(dx0) < step_tol. In this case the previous iteration is returned as the approximation of the root, provided that it satisfies abs(f(x)) < root_tol

  • callback (function, optional) ‚Äì After each iteration callback(x, dx, f(x), iteration) will be called where ‚Äòx‚Äô is the current iteration of the estimated root, ‚Äòdx‚Äô is the step size between the previous and current ‚Äòx‚Äô and ‚Äòiteration‚Äô the number of iterations that have been taken. If the callback function evaluates to True then the routine will end.

Returns:

An approximation for a root of f. If the rootfinding was unsucessful then None will be returned instead.

Find an approximation to a point xf such that f(xf)=0 for a scalar function f using Newton-Raphson iteration starting at the point x0.

  • root_tol (float, optional) ‚Äì The routine ends if abs(f(x)) < root_tol and refine_roots_beyond_tol is False.
